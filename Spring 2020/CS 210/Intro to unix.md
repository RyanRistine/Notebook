# CS 210 Intro to Unix
  ## What is Linux?
    An Operating system
  ## What is an Operating system?
    Software that allows a user to interact with and control hardware
  ## Why Linux
   *  Widely used by
   	* Scientific community
   	* Enterprise data centers
   *  Secure
   *  Open Source
   *  Runs on Most Hardware
   * Windows can run on:
   	* x86
    * Linux can run on:
    	* x86
   	* ARM
    	* MIPS
    	* etc.
